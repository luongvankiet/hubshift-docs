# Hubshift Overview

## What is Hubshift?

Hubshift is a comprehensive, cloud-based NDIS (National Disability Insurance Scheme) case management and CRM platform designed specifically for disability service providers. The platform streamlines operations by supporting client management, worker coordination, appointment scheduling, incident reporting, documentation, and compliance tracking.

Hubshift helps service providers deliver quality care while maintaining compliance with NDIS standards and regulations.

## Main User Roles

The platform supports multiple user roles, each with specific access levels and responsibilities:

### Super Admin / Organisation Admin

- Full system access and configuration
- User management and role assignment
- System settings and configuration
- Complete oversight of all operations

### Support Coordinators / Case Managers

- Client coordination and care planning
- Appointment scheduling and management
- Client goal tracking and progress monitoring
- Documentation and reporting

### Frontline Workers / Support Workers

- View assigned appointments and schedules
- Complete timesheets and expense tracking
- Submit shift notes and documentation
- Report incidents when they occur
- Access client care plans and routines

### Back-Office Staff

- Rostering and scheduling management
- Payroll processing and pay run management
- Financial reporting and invoicing
- Administrative support functions

## High-Level Features

### Client Management

- **Client Profiles**: Comprehensive client information including personal details, NDIS plan information, and medical history
- **NDIS Data**: Track NDIS numbers, plan dates, plan managers, and support coordinators
- **Goals & Objectives**: Set and track client goals with progress monitoring
- **Document Storage**: Secure storage for NDIS plans, medical records, and other important documents
- **Emergency Contacts**: Maintain emergency contact information for each client

### Worker Management

- **Worker Profiles**: Complete profiles for health carers and support coordinators
- **Availability Management**: Track worker availability and schedule appointments accordingly
- **Qualifications**: Manage qualifications, certifications, and training records
- **Performance Tracking**: Monitor appointments, timesheets, and performance metrics

### Appointments & Scheduling

- **Appointment Scheduling**: Create and manage appointments between workers and clients
- **Routine & Task Assignment**: Assign specific routines and tasks to appointments
- **Appointment Swapping**: Allow workers to swap appointments when needed
- **Calendar Integration**: View appointments in calendar format
- **Shift Notes**: Document shift activities and observations

### Incidents & Reports

- **Incident Reporting**: Comprehensive incident reporting system
- **AI-Powered Intake**: AI-assisted incident intake and analysis
- **Compliance Monitoring**: Track incident reporting compliance
- **Analytics & Reporting**: Generate reports and analyze incident trends
- **Geographic Mapping**: Visualize incidents on maps for pattern analysis

### Documentation & Notes

- **Document Storage**: Secure file storage with AWS S3 integration
- **Shift Notes**: Detailed shift documentation and continuation notes
- **Client Notes**: Maintain notes and observations for each client
- **Document Management**: Organize and retrieve documents efficiently

### Notifications

- **Email Notifications**: Automated email notifications for important events
- **SMS Notifications**: SMS alerts via Twilio integration
- **Push Notifications**: Mobile push notifications for real-time updates
- **In-App Notifications**: Notification system within the platform

## Platform Benefits

- **Streamlined Operations**: Centralized platform for all NDIS management needs
- **Compliance**: Built-in features to help maintain NDIS compliance
- **Efficiency**: Reduce administrative overhead with automated workflows
- **Visibility**: Real-time insights into operations and performance
- **Scalability**: Cloud-based infrastructure that scales with your organization
- **Mobile Access**: Mobile-responsive design and dedicated mobile APIs

## Integration Capabilities

Hubshift integrates with various third-party services:

- **Payment Processing**: Stripe integration for payment handling
- **Accounting**: Xero integration for financial management
- **Communication**: Twilio for SMS/WhatsApp, AWS SES for email
- **Storage**: AWS S3 for secure file storage
- **Monitoring**: AWS CloudWatch for logging and metrics
- **AI/ML**: Python-based incident analysis module
